# 자연 치유 LLM Wiki

암 자연치유 커뮤니티 naheal.org의 공개 지식 베이스입니다.

## 구조

```
raw/    # naheal.org 필진이 작성한 원본 아티클 (YAML frontmatter + 마크다운)
wiki/   # LLM(Gemini Flash)이 주제별로 합성한 개념 페이지
```

`raw/`에 아티클이 추가되면 GitHub Actions가 자동으로 `wiki/`를 업데이트합니다.

## 위키 주제

| 주제 | 설명 |
|------|------|
| 맨발걷기/운동 치유 | 접지(earthing) 이론, 맨발 건강, 운동 치유 |
| 식이 요법/영양 치유 | 자연식, 단식, 영양 치유 프로토콜 |
| 명상과 마음 치유 | 마음챙김, 시각화, 스트레스 관리 |
| 약초/자연 요법 | 약초, 자연 요법, 보완 의학 |
| 체험기 | 실제 자연치유 경험담 |
| 의학 정보 | 암 관련 의학 정보 |
| 마음 건강 | 정서적 치유, 심리 지원 |
