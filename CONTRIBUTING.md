# 기여 가이드

## raw/ 아티클 기여

`raw/` 디렉토리에 마크다운 파일을 PR로 제출할 수 있습니다.

### 파일 형식

```markdown
---
articleId: 고유ID (없으면 생략 가능)
title: 아티클 제목
seriesTitle: 시리즈 이름 (선택)
category: 운동|영양|명상|약초|체험기|의학정보|마음건강|일반
topics: [주요 키워드1, 키워드2]
authorName: 작성자 이름
publishedAt: YYYY-MM-DD
source: community
---

## 아티클 본문

마크다운으로 내용 작성...
```

### 주의사항

- 의학적 주장은 출처를 명시하고 신중하게 작성
- 개인 식별 정보 포함 금지
- 광고/홍보 목적 콘텐츠 불가
- 저작권이 있는 내용 무단 게재 금지

## 위키 페이지 직접 수정

`wiki/` 디렉토리의 파일은 GitHub Actions가 자동 생성하므로 직접 수정하면 덮어씌워집니다.  
내용 개선을 원하면 `raw/` 아티클을 추가/수정하거나 issue를 등록해 주세요.
